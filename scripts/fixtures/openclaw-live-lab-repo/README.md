# DevPlat Live Lab Fixture

This repository is created automatically by the DevPlat live E2E lab.

- It is intentionally public so SonarQube Cloud can bind and auto-import it.
- It contains only non-sensitive fixture content.
- It may be deleted at any time by the live-lab cleanup or eviction flow.

## Purpose

The fixture repository exists only to exercise GitHub repository administration,
workflow dispatch, pull request orchestration, and SonarQube Cloud binding from
the DevPlat live lab.
