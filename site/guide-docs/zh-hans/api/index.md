# API 文档

Ironloom 目前暴露一个很小的操作型 API 表面：运行时 HTTP 路由、环境配置、加密设置存储、JSON 模式，以及 crate 级 Rust 边界。

## 章节

- [配置](configuration.md)：环境变量、设置关卡和优先级。
- [运行时 HTTP](runtime-http.md)：健康、就绪、设置和设置提交路由。
- [存储](storage.md)：`.ironloom` 工件和设置存储路径。
- [模式](schemas.md)：已提交的 JSON 模式契约。
- [Crates](crates.md)：Rust crate 所有权边界。
