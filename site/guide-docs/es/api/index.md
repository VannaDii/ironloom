# Docs de API

Ironloom expone hoy una superficie operativa pequeña: rutas HTTP del runtime, configuración de entorno, almacenamiento cifrado de configuración, esquemas JSON y límites Rust a nivel de crate.

## Secciones

- [Configuración](configuration.md): variables de entorno, controles de configuración y precedencia.
- [HTTP del runtime](runtime-http.md): rutas de health, readiness, setup y envío de configuración.
- [Almacenamiento](storage.md): rutas de artefactos y configuración de `.ironloom`.
- [Esquemas](schemas.md): contratos de esquema JSON comprometidos.
- [Crates](crates.md): límites de propiedad de crates Rust.
