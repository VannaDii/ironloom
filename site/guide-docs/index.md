# DevPlat

DevPlat is a Discord-first autonomous software-delivery platform built as a strict native-ESM TypeScript monorepo. It combines auditable platform packages, an adapter-only OpenClaw bridge, generated JSON schemas, and delivery surfaces for packages, containers, charts, and docs.

## What You Get

- contract-first packages with sibling tests and generated schemas
- OpenClaw adapter coverage for the full DevPlat capability surface
- Discord-oriented operator control with GitHub-backed lifecycle state
- Docker and Helm packaging for self-hosted runtime deployment
- SonarCloud-aware CI with strict repository validation
