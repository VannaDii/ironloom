# وثائق API

يعرض Ironloom اليوم سطح API تشغيليا صغيرا: مسارات HTTP لوقت التشغيل، وتهيئة البيئة، وتخزين الإعداد المشفر، ومخططات JSON، وحدود Rust على مستوى الحزم.

## الأقسام

- [التهيئة](configuration.md): متغيرات البيئة، وبوابات الإعداد، والأولوية.
- [HTTP وقت التشغيل](runtime-http.md): مسارات الصحة والجاهزية والإعداد وإرسال الإعداد.
- [التخزين](storage.md): مسارات قطع `.ironloom` الأثرية وتخزين الإعداد.
- [المخططات](schemas.md): عقود مخطط JSON الملتزم بها.
- [الحزم](crates.md): حدود ملكية حزم Rust.
