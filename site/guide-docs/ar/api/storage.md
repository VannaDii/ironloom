# API التخزين

وحده `ironloom-storage` يقرأ أو يكتب مسارات `.ironloom/` مباشرة.

## تخزين القطع الأثرية

تستمر القطع الأثرية تحت جذر حالة وقت التشغيل وتفهرس حسب خيط Discord وعنصر العمل. تستخدم الكتابات ملفا مؤقتا يتبعه rename ذري.

## تخزين الإعداد المشفر

تخزن تهيئة الإعداد المحلي في:

```text
${IRONLOOM_STATE_ROOT}/setup/config.enc.json
```

يحفظ الملف:

- حمولة إعداد JSON مشفرة
- nonce عشوائي
- إصدار تخزين الإعداد

لا تكتب قيم الإعداد النصية إلى القرص. تستخدم كتابات Unix أذونات ملفات للمالك فقط.

## متطلب المفتاح

يجب أن يكون `IRONLOOM_CONFIG_KEY` مادة مفتاح 32 بايت بترميز Base64. يجب أن يتلقى أي pod يعاد إنشاؤه مع PVC نفسه المفتاح نفسه لفك تشفير الإعداد المحفوظ.
